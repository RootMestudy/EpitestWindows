### Prerequisites

- Docker
- Windows 10

# Installation

Git clone this repo on your project root folder

**On your CMD prompt** 
```
install.bat
```

# Usage

Simply ssh through this Docker with
```
run.bat
```
For exit the container, simply type `exit` on the prompt

# Norminette

You can use the Norminette provided by @Gegel85 with
```
norminette -i4 -A
```
